**ElectroHub** is a high-performance, aesthetically pleasing E-Commerce platform inspired by the Apple Store's minimalist design. Built with React and Tailwind CSS, it features smooth transitions, blurred UI elements, and a seamless shopping experience.

---

## 🚀 Key Features

* ** Apple-Inspired UI:** Minimalist design with high-quality typography and spacing.
* **📱 Fully Responsive:** Optimized for Mobile, Tablet, and Desktop with adaptive layouts.
* **🏎️ Featured Scroller:** Custom-built infinite scroller with focus-blur effects for mobile & desktop.
* **🛒 Advanced Cart System:** Real-time cart updates with professional pop-up notifications.
* **⚡ High Performance:** Powered by **Vite** and **React Query** for lightning-fast data fetching.
* **🔔 Toast Notifications:** Smooth, blurred success/error messages for user actions.
* **🔗 Dynamic Routing:** Clean navigation using React Router 6.
