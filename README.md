**ElectroHub** is a high-performance, aesthetically pleasing E-Commerce platform inspired by the Apple Store's minimalist design. Built with the modern MERN-entry stack, it features smooth transitions, blurred UI elements, and a seamless shopping experience.

---

## 🚀 Key Features

* ** Apple-Inspired UI:** Minimalist design with high-quality typography and spacing.
* **📱 Fully Responsive:** Optimized for Mobile, Tablet, and Desktop with adaptive layouts.
* **🏎️ Featured Scroller:** Custom-built infinite scroller with focus-blur effects for mobile & desktop.
* **🛒 Advanced Cart System:** Real-time cart updates with professional pop-up notifications using **Context API**.
* **⚡ High Performance:** Powered by **Vite** and **React Query** for lightning-fast data fetching.
* **📋 Pro Forms:** Efficient form handling and validation using **React Hook Form**.
* **🎨 Premium Icons:** Beautifully integrated UI elements using **HeroIcons**.
* **🔔 Toast Notifications:** Smooth, blurred success/error messages for user actions via **React Hot Toast**.

## 💻 Local Installation

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/fahad-aslam8743/ElectroHub-E-Commerce-Store.git](https://github.com/fahad-aslam8743/ElectroHub-E-Commerce-Store.git)
    ```

2.  **Navigate to the project folder:**
    ```bash
    cd ElectroHub-E-Commerce-Store
    ```

3.  **Install dependencies:**
    ```bash
    npm install
    ```

4.  **Run the development server:**
    ```bash
    npm run dev
    ```
