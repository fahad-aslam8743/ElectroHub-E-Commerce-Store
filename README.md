# 🛒 ElectroHub | Premium E-Commerce Storefront

A sophisticated, high-performance shopping experience built with the modern web stack. ElectroHub features user-specific data persistence, seamless state management, and a highly polished UI designed for both desktop and mobile users.

**🔗 Live Demo:** [electro-hub-e-commerce-store.vercel.app](https://electro-hub-e-commerce-store.vercel.app)

## ✨ Key Features

* **Authentication System:** Complete Login/Signup flow with secure session handling.
* **User-Specific Carts:** Items are tied to your unique `user_id`, ensuring your shopping list is never lost.
* **Interactive UI:** Framer Motion animations for smooth transitions and Lucide icons for a modern feel.
* **Smart Forms:** React Hook Form integration for real-time validation and error prevention.
* **Toast Notifications:** Instant feedback via React Hot Toast for a professional "app-like" experience.
* **Product Details:** Dynamic routing for product pages and interactive pop-ups for quick views.

---

## 🛠️ Tech Stack & Tools

### **Core Frameworks**
* **React + Vite:** For a lightning-fast development and build environment.
* **React Router DOM:** Handling complex navigation and dynamic product routing.

### **Data & State**
* **TanStack Query:** Managing server state, caching, and efficient data fetching.
* **React Hook Form:** For high-performance, flexible form management.

### **UI & Styling**
* **Tailwind CSS:** For custom, utility-first styling.
* **Framer Motion:** For advanced gestures and entrance animations.
* **Lucide React:** Beautiful, consistent iconography.
* **React Hot Toast:** Elegant, non-blocking notifications.

## 🚀 Getting Started

1.  **Clone the Repo**
    ```bash
    git clone [https://github.com/your-username/electro-hub.git](https://github.com/your-username/electro-hub.git)
    ```
2.  **Install Dependencies**
    ```bash
    npm install
    ```
3.  **Launch**
    ```bash
    npm run dev
    ```

## 🛡️ Architecture & Logic
* **State Persistence:** Cart logic is synchronized between the local state and the database, ensuring no data loss on refresh.
* **Responsive Logic:** Built with a "Mobile-First" philosophy to ensure accessibility across all device types.
* **Clean Code:** Modular component structure for high maintainability and scalability.
